# 📱 QR Code Generator

A simple Python script that generates QR codes from user input and saves them as an image.

---

## ✨ Features

✅ Generate QR codes instantly

✅ Save QR code as PNG image

✅ Automatically open the generated QR code

✅ Lightweight and beginner-friendly

---

## 📦 Requirements

Install the required libraries:

```bash
pip install qrcode pillow
```

---

## 🚀 Usage

Run the script:

```bash
python qr_generator.py
```

Enter the text, URL, or any data you want to convert into a QR code:

```text
QR: https://github.com/DhyeyLukhi
```

The script will:

1. Generate a QR code 📱
2. Save it as `newqr.png` 💾
3. Open the image automatically 🖼️
